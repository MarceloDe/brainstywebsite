# **App Name**: Brainsty Health Navigator

## Core Features:

- Landing Page with Mission Pillars: Interactive sections on the landing page that illustrate Brainsty's mission pillars (#Translate, #Integrate, #Delivery) with real-world scenarios. Long-form, vertical scrolling design.
- Wefella AI Concierge Chat: A web chat interface (similar to ChatGPT) powered by a custom-prompted OpenAI model that acts as an expert healthcare concierge. Access restricted to logged-in users.
- Curated Research Tool: A 'smart search' tool that uses Google's Gemini LLM tool to find relevant scientific papers, summarize the information, and present it in an easy-to-understand chat format. Requires user login to view results.
- Services Overview Page: Details the specific services offered by Brainsty (informed clinical decision support, insurance navigation, etc.) as described in the provided documentation.
- User Authentication System: A secure user registration and login system to control access to the AI Concierge and Curated Research tool.

## Style Guidelines:

- Primary color: Soft, desaturated blue (#A0B1BA) to evoke trust and tranquility, inspired by the 'clarity' indicated in the company aesthetic.
- Background color: Light gray (#F0F2F5), a desaturated, bright tint of the primary color, for a clean, professional backdrop.
- Accent color: Lavender (#D0B4DE), an analogous color to the primary, for interactive elements and highlights.
- Headline font: 'Belleza', sans-serif. Body font: 'Alegreya', serif. Pairing for a blend of contemporary and trustworthy feeling.
- Line art icons inspired by neurology, connections, and clarity, reflecting the company's core identity, based on provided images.
- Clean, vertical-scrolling layout with intuitive tab/menu navigation. Fully mobile-responsive design for a seamless experience on all devices.
- Subtle transitions and animations to enhance user engagement and provide feedback, reinforcing a sense of dynamic interaction.
